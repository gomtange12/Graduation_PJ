트위치 채팅을 가져오는건 JSON으로 파싱한 데이터가아니라
IRC 라는 실시간 인터넷 채팅 프로토콜을 사용하여 서버 : 서버 로 채팅을 할 수 있게 하는 것이다.

1. https://twitchapps.com/tmi/에서 토큰을 발급받음 

2. IRC 클라이언트 설정에서

서버: irc.chat.twitch.tv
포트: 6667
닉네임: 트위치 ID (1k1000)
서버 비밀번호: 저기 저 토큰
채널: #채널스트리머 아이디

인터넷 릴레이 챗(Internet Relay Chat, IRC)은 실시간 채팅 프로토콜로, 여러 사용자가 모여 대화를 나눌 수 있다.


IRC는 TCP 기반의 공개 네트워크 프로토콜 지원

대부분의 서버에서는 6667/TCP 포트를 사용한다.
